# EnumWindow-T

I do not plan to provide an English version of this README file or the program.

这个项目是对 WindowContorlEx 项目（未公开）的全面升级。

这个项目的目标是提供类似 spy++ 的窗口查找功能，同时提供一些实用的操作。

> [!NOTICE]
> 使用本程序的各个模块需要可能不同水平的电脑知识。作者不对因不当操作造成的任何损失负责。

# 功能介绍
- 通过拖动光标查找窗口
- 通过窗口标题和类名查找窗口
- 黑名单过滤
- 实时跟踪窗口参数

# 使用案例

## 查找弹窗来源

> 通过拖动光标查找窗口，定位弹窗程序。


# 注意事项

程序扫描出的窗口列表是类似于“快照”的一个东西，并不会实时更新，也不保证扫描期间创建或销毁的窗口能被正确处理。

程序的窗口坐标修改功能目前对于子窗口有bug，请勿使用。
